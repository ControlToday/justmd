# justmd
Simple markdown editor based on electron.

# run
npm install
npm start

# build
before build:
set ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/

for windows package:
npm run package-x86
or
npm run package-x64

for osx package(must use osx host):  
npm run package-osx
